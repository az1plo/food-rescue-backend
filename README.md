# Názov témy

Food Rescue Platform

## Stručný popis témy

Food Rescue Platform je platforma zameraná na znižovanie plytvania potravinami prostredníctvom evidencie, publikovania a rezervácie prebytočných potravín z reštaurácií a obchodov. Systém umožňuje registráciu používateľov, publikovanie ponúk, ich vyhľadávanie a rezerváciu na vyzdvihnutie. Súčasťou riešenia je aj základná administrácia systému a evidencia stavu ponúk a rezervácií. Cieľom aplikácie je efektívnejšie využitie nepredaných potravín a podpora udržateľnej spotreby.
