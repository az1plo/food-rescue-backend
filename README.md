# Názov témy

Food Rescue Platform

## Stručný popis témy

Food Rescue Platform je webová aplikácia určená na znižovanie plytvania potravinami prostredníctvom evidencie, publikovania a rezervácie prebytočných potravín z reštaurácií a obchodov. Systém umožňuje registráciu a správu používateľov, publikovanie ponúk, ich vyhľadávanie a rezerváciu na vyzdvihnutie. Súčasťou riešenia je aj základná administrácia obsahu a evidencia stavu ponúk a rezervácií. Cieľom aplikácie je efektívnejšie prerozdelenie nepredaných potravín a podpora udržateľnej spotreby.
